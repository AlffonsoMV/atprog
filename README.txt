## Guide d'Utilisation

Le programme se compose de deux exécutables principaux :

1. **Google_tests** : Ce sont des tests implémentés pour notre projet en utilisant la bibliothèque de Google. L'ensemble comprend un total de 16 tests conçus pour assurer la qualité et le bon fonctionnement du projet.

2. **Projet Principal** : Celui-ci se lance directement via `main.cpp`. Au démarrage, une fenêtre avec une interface graphique s'ouvre, permettant de :

   - Sélectionner la taille du plateau.
   - Choisir le solveur.
   - Modifier les paramètres d'entrée spécifiques pour chaque solveur.

   Une fois les paramètres configurés à votre satisfaction, vous pourrez cliquer sur "Solve". Cela lancera l'exécution du solveur et affichera la solution à l'écran.

### Étapes pour Exécuter le Projet Principal

1. Compilez et exécutez `main.cpp`.
2. Dans l'interface graphique, configurez la taille du plateau.
3. Sélectionnez le type de solveur que vous souhaitez utiliser.
4. Ajustez les paramètres d'entrée selon vos besoins.
5. Cliquez sur "Solve" pour voir la solution générée.

### Informations Supplémentaires

- Assurez-vous qu'Imagine++ est installé et que vous utilisez QT Creator.
- Notez que lorsque vous exécutez l'algorithme de Backtracking, la même solution apparaît toujours. Cela se produit parce que cet algorithme est le seul algorithme implémenté qui est déterministe.
- Vous pouvez trouver une version améliorée de ce README à https://github.com/AlffonsoMV/atprog/
